# グランドライブ 調査メモ

## 攻略メモ

### ウララのローテーション

ファンボそこそこ（安定させるには85ぐらい）積んだウララなら全日本ジュニア優駿と告知ライブ2回でファン数12000に届き、クラシックの目標全達成。スキルPt足りないなら南部杯JBCチャンピオンズカップ出走も可能。アオハルで朝日杯突撃してたのと比べれば遥かに楽になった。

### 限界突破の影響

スキルPtはこれまで、1Pあたりのレースへの影響度が少なかったが、限界突破分の上昇量半減の影響を受けないため、相対的に価値が上昇。ただ、ステ1=スキルPt1とまでは言えない気がする。現段階ではスピ以外限界突破は狙わないだろうし。

以下では計算しやすいように、*スキルPt1=ステ1*として計算。

### トレーニング上昇量 vs 友情トレーニング獲得量

* トレ上昇量の効果
  * トレ上昇量+1 = 対象ステ+1×(100+友情トレ獲得量)/100
* トレ上昇量有効率
  * スキルPt: 100%
  * スピ: スピ賢編成の場合ほぼ100%、スタ必要なら少し下がる
  * 賢さ: 序盤は連打するが、他の友情が出せるようになれば下がる
  * スピとスキルPt以外、友情トレ獲得量の方が仕事する機会が多そう
* 友情トレ獲得量の効果
  * キタサン単独スピLv3　→　友情トレ獲得量+5% = スピ+0.85、パワ+0.4、スキルPt+0.15
  * キタサンタキオン編成外3人スピLv5　→　友情トレ獲得量+5% = スピ+1.65、パワ+0.9、スキルPt+0.5
  * ファイン単独賢さLv5　→　友情トレ獲得量+5% = スピ+0.2、賢さ+0.9、スキルPt+0.3
  * ファインカレン編成外3人賢さLv5　→　友情トレ獲得量+5% = スピ+0.7、賢さ+2.15、スキルPt+0.5
  * 友情トレ獲得量+5%で、ステ+1.4～3.0程度（時期が遅いほど数値が増える）
    * トレ選択AI作って頑張れば期待値出せるけど大変なので割愛
* 友情トレ発生率
  * 例として、スピ:キタサンタキオン、パワ:ライスエル、賢さ:ファインカレンで計算
    * 得意率アップ0　→　友情トレ発生率83%
    * 得意率アップ30　→　友情トレ発生率89.08%
  * 友情トレ獲得量は約85%の確率で仕事する
* 1回のトレーニングあたりのステ上昇
  * 条件
    * トレ上昇量+1=ステ+1.3（友情トレ獲得量+30%）、友情トレ獲得量+5%=ステ+2で試算
  * 結果
    * スキルPt上昇量+1: 1.3
    * スピ上昇量+1(有効率95%想定): 1.235
    * 賢さ上昇量+1(有効率60%想定): 0.78
    * パワ上昇量+1(有効率30%想定): 0.39
    * 友情トレ獲得量+5%: 1.7
* 有効な期間
  * トレーニング上昇量: 購入直後から有効
  * 友情トレ獲得量: 次のライブから有効
* 購入優先判断
  * スキルPt上昇量+1 vs 友情トレ獲得量+5%
    * スキルPt上昇量の有効期間が約1.3倍長ければ同等
    * シニア7月～12月ならばスキルPt上昇量が上
    * ただし、スキルPt1をステ1と同等に扱って良いかという問題あり
  * スピ上昇量+1 vs 友情トレ獲得量+5%
    * シニア7月～12月前半ならばスピ上昇量が上
  * 賢さ上昇量+1 vs 友情トレ獲得量+5%
    * シニア7月～10月ならば賢さ上昇量が上
  * パワ上昇量+1 vs 友情トレ獲得量+5%
    * シニア7月ならばパワ上昇量が上
  * 実際は目標レースとお休みの分終了時期が繰り上げ
  * スピ+2以上と、おそらくスキルPtの+2以上は、友情トレ獲得量+5%より優先
* ジュニア期
  * クラシック前半で絆ゲージが不足するなら、トレーニング上昇量が有利
  * ただ、絆を最速で溜めて友情を狙う動きになるので、おそらく友情トレ獲得量が上

### レッスン優先順位

#### 楽曲更新を考慮せず計算

* テクニックの価値
  * 共通
    * 次の告知ライブ/グランドライブ後に、スキルPtがレッスン回数×5
    * ファイナルズ期に獲得した場合は入らない（シニア12月は使い切るのが得）
  * ステ上昇（計算上、スキルPt含む全ステの価値は平等とする）
    * +5: 10パフォ、ステ+5、スキルPt+5　→　1パフォ=1ステ
    * +8: 16パフォ、ステ+8、スキルPt+5　→　1パフォ=0.81ステ
    * +12: 24パフォ、ステ+12、スキルPt+5　→　1パフォ=0.71ステ（ファイナルズ期は1パフォ=0.5ステ）
    * F期: 24パフォ、ステ+12　→　1パフォ=0.5ステ
  * スキルヒント
    * Lv1は欲しければ引かざるを得ない
    * Lv2以降、スキルの基本コストを150、必要なスキルの割合をXとすると、Lv1と比較して
      * Lv2: パフォ+10、スキルPt+15　→　1パフォ=(1.5×X)ステ　→　X>0.333…でステ上昇より上
      * Lv3: パフォ+15、スキルPt+30　→　1パフォ=(2.0×X)ステ　→　X>0.25でステ上昇より上
      * 実際、候補スキルのうち25%以上を取りたいケースは無いと思う（確認はしてない）
  * 体力
    * 以下の計算より、体力70=ステ87と仮定
      * 体力70程度回復することで、お休みを1回トレーニングに変更できる
      * キタサンファイン編成外2人（そんなに上振れじゃない）スピLv5、友情獲得+50、スピパワスキルPt上昇量+3で、合計87上昇
    * 体力+20: パフォ25、ステ24.9相当、スキルPt+5　→　1パフォ=1.20ステ（F期:1パフォ=0.99ステ）
    * 体力+30: パフォ30、ステ37.3相当、スキルPt+5　→　1パフォ=1.41ステ（F期:1パフォ=1.24ステ）
    * 体力+40: パフォ35、ステ49.7相当、スキルPt+5　→　1パフォ=1.56ステ（F期:1パフォ=1.42ステ）
    * ステ上昇より効率が良い
    * 低体力で強い配置が来た時の保険としてキープもあり
    * ただし、上振れは賢さや友人で回復してお休みを踏まないパターンなので、取らずに済むならそっちの方が良い

#### 楽曲更新を考慮して計算

* 楽曲レッスンが登場するタイミングは、告知ライブ後の、2,5,8,13,19……回目の更新（ジュニアのみ2,5,9……）
  * テクニック表示状態で告知ライブを迎えた場合、ライブ後にテクニック→テクニック→楽曲の順で獲得
  * 楽曲表示状態で告知ライブを迎えた場合、ライブ後に楽曲→テクニック→楽曲の順で獲得
* 獲得楽曲数の影響
  * ステータス
    * 告知ライブ: 新曲3曲以上で全ステ+10、それ以下で全ステ+3
    * 通常グランドライブ: 新曲3曲以上で全ステ+12、それ以下で全ステ+5
    * 特別グランドライブ: 新曲3曲以上で全ステ+15、それ以下は未確認
  * スキルPt
    * 告知ライブ/グランドライブ後に楽曲数×20（イベント獲得楽曲は除く）
  * スキルヒント
    * シニア6月ライブ後に16曲→夢の途中
    * シニア11月後半行動前16曲→選択肢に応じた金or白スキル
    * シニア12月後半行動前18曲→キミと勝ちたい
* テクニックは、ジュニア→クラシック→シニアと進むにつれ、効果が上がり、必要パフォが増える
  * 効率の上がる体力を除き、安い方が更新できて強い
* 楽曲表示して告知ライブ vs 楽曲獲得して告知ライブ
  * 楽曲獲得した場合の変化
    * 次の6か月でライブボーナスを1つ多く使える
      * 効率計算
        * 上の友情トレ獲得量+5%の計算よりトレ1回あたりステ+1.7、ただし時期によって変わるので補正
        * 6か月にトレ9回（3/4がトレ）とする
        * 更新には各時期のステ上昇を使用
        * ジュニア
          * 効果量: 絆不足やトレLv不足で効果が大きく下がる　→　ステ+1.0で計算
          * 更新コスト: パフォ10（ステ+5）
          * 合計ステ上昇: 1.0×9+5=14.0
          * 効率: 1パフォ=1.4ステ
        * クラシック
          * 効果量: ステ+1.7で計算
          * 更新コスト: パフォ16（ステ+8）
          * 合計ステ上昇: 1.7×9+8=23.3
          * 効率: 1パフォ=1.45ステ
        * シニア
          * 効果量: ステ+2.0で計算
          * 更新コスト: パフォ24（ステ+12）
          * 合計ステ上昇: 2.0×9+12=30.0
          * 効率: 1パフォ=1.25ステ
      * パフォ効率はステ上昇を取るより高い
    * 告知ライブの獲得スキルPt+5？
    * 更新コストがテクニック1つ分多くかかる
      * パフォがギリギリの場合、更新コストが増えて楽曲取れなくなるリスクがある
    * 友情トレ獲得量+5%なら獲得して告知ライブで良さそう、他は安定を取りたい
* 楽曲表示して告知ライブ vs 前の楽曲獲得した時点で止める
  * 今回は8回目更新分まで(3～4曲)獲得し、次回は4曲獲得する場合
    * 楽曲表示して告知ライブ
      * 今回: 13回目表示で止める　→　前回分1回、今回分の楽曲3回、テクニック9回
      * 次回: 2,5,8回目更新分を獲得　→　今回分の楽曲1回、次回分の楽曲3回、テクニック5回
    * 前の楽曲獲得した時点で止める
      * 今回: 8回目獲得で止める　→　前回分1回、今回分の楽曲3回、テクニック5回
      * 次回: 2,5,8,13回目更新分を獲得　→　今回分のテクニック1回、次回分の楽曲4回、テクニック9回
    * 後者の場合、テクニック1回分更新コストが増えるので、前者が有効
  * 13回目まで獲得する場合、必要合計曲数を考えると、19回目表示はおそらく非効率（TODO:計算）
* 楽曲獲得後、いくつかテクニックを獲得し、次の楽曲が出る前に止めるのは非効率
  * でも次まで行けると思ったら事故ることが割とよくある
* 基本方針
  * ジュニア期: できるだけステ上昇（安い）を選んで更新する、友情なくて偏りやすいので、楽曲の必要パフォ確認
  * クラシック期: ステ上昇よりヒントLv1が安いので、できればヒント取りたい
  * シニア期: 同上、楽曲数に余裕があれば体力が取れると得
* 楽曲獲得タイミング例
  * ジュニア9回、クラシック前半13回、クラシック後半13回、シニア前半13回（毎回4曲分更新する）
    * シニア6月の夢の途中が不要の場合、目標が厳しい時期は8回でも可
  * シニア後半は取れるだけ取る
